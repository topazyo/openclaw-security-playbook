# Security Scripts

This directory contains automation scripts for various security tasks.

## Directories

- `monitoring/`: Behavioral monitoring and anomaly detection
- `supply-chain/`: Skill integrity and supply chain security
- `verification/`: Security verification and compliance checking
- `discovery/`: Shadow AI discovery and inventory

## Usage

All scripts include built-in help:

```bash
./scripts/monitoring/anomaly_detector.py --help
```
