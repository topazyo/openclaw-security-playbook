# Security Scripts

This directory contains automation scripts for various security tasks.

## Directories

- `incident-response/`: Incident containment, notification, and analysis automation
- `forensics/`: Evidence collection, credential scoping, timeline building, and hash-chain verification
- `monitoring/`: Behavioral monitoring and anomaly detection
- `supply-chain/`: Skill integrity and supply chain security
- `verification/`: Security verification and detection validation

## Usage

All scripts include built-in help:

```bash
./scripts/monitoring/anomaly_detector.py --help
```
